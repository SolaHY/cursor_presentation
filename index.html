🏃基本的な使い方

  1. Tab補完でスピードアップ
    コメントや関数名を書いた後、Tabキーを押すと自動でコードが生成されます
  2. Cmd+K で部分編集
    修正したいコードを選択 → Cmd+K → 「このコードにエラーハンドリングを追加して」
  3. Cmd+L でAIチャット
    「この関数の動作を説明して」「なぜエラーが出るの？」など自由に質問
  4. @でファイル指定
    「@app.py このファイルのバグを見つけて」のように特定ファイルを参照

❓効果的な質問パターン

❌良くない質問

  • 「動かない」
  • 「エラーが出る」
  • 「どうすればいい？」

✅良い質問

  • 「NameError: name 'x' is not definedというエラーの解決方法は？」
  • 「forループで配列の要素を1つずつ表示したい」
  • 「このif文の条件式が理解できない」
